**[ НАЗВАНИЕ ПРОЕКТА ]**

“[Alfa-ИВР] Кейс №12 - HR-решение для проведения ИТ-собеседований”

**[ ПРОБЛЕМНОЕ ПОЛЕ ]**

Пользователю необходимо найти информацию о прохождении собеседований, а для этого ему необходимо видеть на экране статьи от пользователей с данной информацией. Пользователю необходимо подготовиться к предстоящему тестовому испытанию, а для этого ему необходимо зайти в раздел с тестовыми заданиями. Пользователю необходимо найти тестовые задания на определённую тематику, а для этого он в фильтрах выбирает те параметры, которые ему наиболее интересны. Пользователю необходимо узнать отзывы и статьи о компаниях, а для этого на экране отображается вкладка со списком компаний, на которые были написаны отзывы/статьи. Компании необходимо провести конкурс для отбора на стажировку, а для этого необходимо добавить раздел со статистикой прохождения опубликованного конкурса.

**[ ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ ]**

Программный продукт будет предоставлять следующие возможности:

- Фильтрация статей по выбранным пользователем параметрам
- Публикация статьи/отзыва
- Регистрация/Аутентификация
- Прохождение тренировочных тестовых заданий
- Публикация тренировочных тестовых заданий от пользователей или компаний
- Создание конкурса компанией
- Возможность создавать раздел от лица пользователя
- Вывод статистики по созданному компанией конкурсу
- Экспорт результатов в различные форматы
- Админ панель для модерации сайта

**[ ПОХОЖИЕ / АНАЛОГИЧНЫЕ ПРОДУКТЫ ]**

Анализ 3 программных продуктов, которые максимально приближены к заданному функционалу, показал, что:

- LeetCode: не обладает возможностью публикации полноценных конкурсов от лица компании, не обладает возможностью создавать личные разделы пользователей
- Reddit: отсутствие раздела с тренировочными тестовыми заданиями, отсутствие возможности создания конкурса и просмотра статистики по нему у компаний
- Lolzteam: отсутствие раздела с тренировочными тестовыми заданиями, сортировка отзывов и статей по компаниям
